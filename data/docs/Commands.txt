## 🎉 New Features in v2.1

### 1. Hybrid Search (BM25 + Dense + RRF)
- Better retrieval accuracy (+16-19%)
- Combines semantic + keyword search
- Enabled by default

### 2. Enhanced LLM Providers
- **vLLM**: 10x faster local inference
- **Groq**: 500+ tokens/sec (fastest cloud)
- **Together AI**: 100+ model options

### 3. Query Understanding Pipeline
- Automatic query classification
- Query rewriting (typos, abbreviations)
- HyDE for better semantic matching

### 4. Feedback Service
- Thumbs up/down ratings
- Implicit feedback tracking
- Quality analytics

### 5. WebSocket Real-time Chat
- Token streaming
- Typing indicators
- Cancel generation

## 🚀 Quick Start

### 1. Install Dependencies
```bash
cd /home/master/Documents/My_Courses/GenAI_Projects/Enterprise_Chatbot_002
pip install -r requirements.txt
```

### 2. Ingest Documents
```bash
python ingest.py
```

### 3. Start Server

uvicorn main:app --reload --host 0.0.0.0 --port 8000
```bash

```

### 4. Access Application
1. Open browser
2. Go to: **http://127.0.0.1:8000/frontend/index.html**
3. Sign up for a new account
4. Start chatting!

## 📡 API Endpoints

### New Endpoints in v2.1

#### Feedback
```bash
# Submit thumbs up/down
curl -X POST http://localhost:8000/api/feedback/explicit \
  -H "Content-Type: application/json" \
  -d '{"conversation_id":1,"message_id":1,"is_positive":true}'

# Get feedback stats
curl http://localhost:8000/api/feedback/stats
```

#### WebSocket (Real-time Chat)
```javascript
const ws = new WebSocket('ws://localhost:8000/ws/chat?token=YOUR_TOKEN');

ws.onmessage = (event) => {
  const msg = JSON.parse(event.data);
  console.log(msg.type); // 'typing_start', 'token', 'chat_response'
};
```

## 🐛 Troubleshooting

### Tests Failing
```bash
# Clear cache and rerun
rm -rf .pytest_cache __pycache__
python -m pytest tests/test_enhancements.py -v --cache-clear
```

### Import Errors
```bash
# Reinstall dependencies
pip install -r requirements.txt --force-reinstall
```

### Server Not Starting
```bash
# Check if port 8000 is in use
lsof -i :8000

# Kill existing process
pkill -f "uvicorn main:app"

# Restart
```
