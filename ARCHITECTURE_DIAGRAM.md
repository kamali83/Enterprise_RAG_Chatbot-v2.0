## Request Flow Diagram

### Standard RAG Query Flow

```
User Query
    │
    ▼
┌─────────────────────────────────┐
│  1. API Endpoint (/ask)         │
│     • Authenticate (JWT)        │
│     • Rate Limit Check          │
│     • Request Logging           │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  2. Query Understanding         │
│     • Classify Query Type       │
│     • Rewrite (fix typos)       │
│     • Generate Enhanced Query   │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  3. Cache Check (Redis)         │
│     • Semantic Similarity       │
│     • Cache Hit? → Return       │
└──────────────┬──────────────────┘
               │
               │ Cache Miss
               ▼
┌─────────────────────────────────┐
│  4. Hybrid Retrieval            │
│     ├─ Dense Retrieval          │
│     └─ BM25 (Sparse) Retrieval  │
│           │         │            │
│           └────┬────┘            │
│                ▼                 │
│        RRF Fusion                │
│        Ranked Results            │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  5. Context Building            │
│     • Select Top K Docs         │
│     • Add Metadata Headers      │
│     • Truncate if needed        │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  6. LLM Generation              │
│     • Select Provider           │
│     • Create Prompt             │
│     • Generate Answer           │
│     • Stream Tokens             │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  7. Response & Caching          │
│     • Cache Result (Redis)      │
│     • Add Sources               │
│     • Return to User            │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  8. Feedback Collection         │
│     • Track Implicit Signals    │
│     • Wait for Explicit Input   │
│     • Update Analytics          │
└─────────────────────────────────┘
```
